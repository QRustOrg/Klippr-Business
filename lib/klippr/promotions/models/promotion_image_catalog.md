# Promotion Image Catalog

Business stores the selected promotion image as `imageKey`. Consumer must keep
the same keys and resolve them to equivalent bundled assets.

Fallback key: `comida_hamburguesas`.

Allowed keys:

- `comida_pizza`
- `comida_hamburguesas`
- `comida_pollo_frito`
- `comida_ceviche`
- `salud_pastillas`
- `salud_medicamentos`
- `entretenimiento_cine`
- `entretenimiento_bolos`
- `entretenimiento_bares`
- `entretenimiento_cibercafe`
- `deportes_futbol`
- `deportes_volley`
- `deportes_basket`
